## Project Overview

**Vector Manufacturing and Advertising PLC** is a frontend-focused web platform designed to provide a modern and interactive digital presence for the company. The platform showcases the company’s services, products, and portfolio in a user-friendly and visually appealing manner. 

The **frontend is fully built using React and CSS**, while the backend logic is still under development to handle dynamic content, data management, and full-stack functionality. This project highlights my skills in frontend development, responsive design, and structured project planning, while laying the foundation for future backend integration.

---

## Features

- **Modern and responsive frontend design**
- **Service and product showcase sections**
- **Interactive portfolio display**
- **User-friendly navigation**
- **Mobile-friendly and responsive layout**
- **Planned backend integration for dynamic content**
- **Well-structured project architecture**
- **Ongoing development for full-stack capabilities**
